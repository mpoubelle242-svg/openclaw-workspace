# 🦞 OpenClaw Workspace

Workspace personnel pour [OpenClaw](https://docs.openclaw.ai) — assistant IA personnel auto-hébergé.

## Structure

```
├── AGENTS.md       # Configuration des agents
├── SOUL.md         # Personnalité de l'assistant
├── USER.md         # Profil utilisateur
├── MEMORY.md       # Mémoire longue durée
├── HEARTBEAT.md    # Tâches périodiques
├── IDENTITY.md     # Identité de l'agent
├── TOOLS.md        # Outils & accès
└── memory/         # Journaux quotidiens
```

## Agents

| Agent | Rôle |
|-------|------|
| `main` | Superviseur (Le Patron) |
| `whatsapp` | Messagerie WhatsApp |
| `research` | Recherche web |
| `files` | Gestion de fichiers |
| `email` | Email |
| `calendrier` | Calendrier |

## Usage

Ce workspace est géré automatiquement par OpenClaw. Les fichiers sont synchronisés avec le gateway.

## Sécurité

> ⚠️ Ce dépôt contient des données personnelles. Ne pas rendre public.
